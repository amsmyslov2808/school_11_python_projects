## Алгоритм 

### Шифрования
1. Задать алфавит для шифрования
2. Спросить у пользователя текст для шифрования
3. Задать шаг шифрования (+Нормализовать шаг шифрования)
4. Найти в алфавите текущую букву из текста и прибивать к её идексу шаг шифрования, если новый индекс для буквы выходит за пределы алфавита справа, то надо начинать считать индекс с начала алфавита
5. Сформировать по букве новую строчку из зашифрованного текста, если текущая буква из текста отсутсвует в алфавите, то переносим эту букву в шифрованный текст как есть
6. Вывести зашифрованный текст на экран

### Дешифрования
Алгоритм полностью повторяет алгоритм шифрования, только в 4-м пункте из индекса текущей буквы вычитают ключ шифрования, если новый индекс для буквы выходит за пределы алфавита слева, то надо начинать считать индекс с конца алфавита

Если текущая буква из текста отсутсвует в алфавите, то переносим эту букву в дешифрованный текст как есть.
